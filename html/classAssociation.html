Detailed Description

The Association Class.

An Association consists in a group of associates. It must have a name and a fund that is used to help promote event hosted by it's associates. An Association also has a group of Science Areas and each one of these has a Subarea. Each year, the associates must pay the Association's annual pay, which reverts to the fund. Finally, the Association has a Network used by it's associates, in which they can send and receive emails.
